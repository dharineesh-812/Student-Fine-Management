# 🎓 Student Fine Management System

## 📌 Overview
The Student Fine Management System is a console-based Java application designed to manage student fine payments efficiently. 

The system implements role-based access control and allows different users (Admin, Cashier, and Student) to perform specific operations based on their role.

This project is developed using core Java and Object-Oriented Programming principles without using a database. Data is stored temporarily using ArrayList collections.

---

## 🚀 Features

### 👨‍💼 Admin
- View all payments
- Add payment
- Update payment
- Delete payment
- Filter payments by fine type
- Filter payments by student

### 💰 Cashier
- Add payment
- Update payment
- Delete payment

### 🎓 Student
- View profile
- Edit profile

### 🔐 Authentication
- User Registration
- User Login
- Role-based access control
